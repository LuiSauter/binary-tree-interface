## Requerimientos básicos
**Python** y **Pip**, verificar si está instalado
```bash
python --version
pip3 --version
```

# Instalación
Virtual env
```bash
# entorno virtual
pip3 install virtualenv
```
Crear entorno virtual
```bash
# entorno virtual
python -m virtualenv venv
```
Activar entorno virutal
```bash
# Linux o Mac
source venv/bin/activate
# Windows
.\venv\Scripts\activate
```
